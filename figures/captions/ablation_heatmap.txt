Figure 6: Ablation Study Results. Detection matrix showing the impact of removing individual AEGIS components (§5.4). Full AEGIS detects all attack types. Removing volume constraints allows volume-based exfiltration to go undetected. Removing sensitive file policies allows unauthorized file access. Removing covert channel detection bypasses out-of-band communication monitoring. Removing injection detection enables tool injection attacks. The minimal configuration detects nothing, confirming that all components are necessary for comprehensive protection.
