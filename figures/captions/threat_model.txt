Figure 2: Hijacked Agent Threat Model. A legitimate AI agent (left) possessing valid credentials—Kerberos ticket, SSH key, and RBAC badge—is targeted by four injection vectors (center): filesystem-mediated injection through shared storage, co-location attacks via shared compute nodes, supply chain compromise through trusted tools, and network-based injection. Once hijacked (right), the agent exfiltrates data through an encrypted LLM API channel that is opaque to traditional DLP and IDS systems. The attack preserves all original credentials while operating under the attacker's control.
