Figure 3: HPC Injection Attack Surfaces. Four attack vectors exploit HPC shared infrastructure: (Panel 1) Filesystem-mediated attack where User A poisons an HDF5 file on shared Lustre/GPFS storage, which User B's agent reads and trusts; (Panel 2) Co-location attack where two users share a compute node and /tmp directory, enabling local injection; (Panel 3) Supply chain attack where a compromised data_converter tool produces malicious output trusted by the agent; (Panel 4) Coordinated multi-node attack where Agent 1 writes to /.cache/ on Node 1, and Agent 3 on Node 3 reads the payload and exfiltrates via the LLM API.
