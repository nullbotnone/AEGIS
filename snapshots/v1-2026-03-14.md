## Key Decisions

- Core threat: hijacked authorized agent via prompt injection
- Four HPC-specific attack vectors: filesystem-mediated, co-location, supply chain, coordinated multi-agent
- Behavioral attestation as new security primitive: provable, constraint-based, runtime
- Attestation via eBPF syscall interception on compute nodes
- Containment via Slurm REST API (rate-limit → isolate → suspend → terminate)
- Implementation integrated into §4 (not separate section)
