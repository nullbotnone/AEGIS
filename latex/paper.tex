% Abstract
\begin{abstract}
AI agents are beginning to automate data analysis, orchestration, and simulation steering in HPC environments, but they introduce a security problem that existing cluster defenses do not address: the hijacked authorized agent. An adversary can subvert an agent through prompt injection, poisoned shared data, compromised tools, or multi-agent coordination, then exfiltrate sensitive data through legitimate encrypted LLM API traffic. We propose behavioral attestation as a runtime defense for this setting. Our system, AEGIS, compiles task-derived behavioral constraints, monitors agent actions with \ebpf-based syscall interception, verifies evidence against constraint profiles, correlates behavior across agents, and triggers \slurm{}-level containment. On the real attestation path, AEGIS detects all four implemented attack classes with sub-second median latency at a 1\,s interval and low measured verifier overhead. Controlled experiments show full attack-suite coverage, stronger detection than representative baselines, and no observed false positives on the evaluated benign workflows.
\end{abstract}
